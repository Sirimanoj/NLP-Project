# Graph IR Frontend (Vercel)

Static frontend for your Graph IR project.

## Deploy on Vercel

1. Push repo to GitHub.
2. In Vercel: **Add New... -> Project**.
3. Select this repository.
4. Set **Root Directory** to `graph_ir_frontend`.
5. Framework preset: **Other** (static).
6. Deploy.

## Use after Deploy

- Open the deployed Vercel URL.
- Paste your Render backend URL in **Render API Base URL**.
- Click **Load Corpus**, then **Run Retrieval**.
